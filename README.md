# des-demo

a demo application of DES (Data Encryption Standard)

对称密钥加密算法DES的演示程序

使用命令行输入

```
SYNOPSIS
        Untitled-1.exe [-f <input file>] [-k <key>] [-m <mode>]

OPTIONS
        -f, --filepath <input file>
                    file path

        -k, --key <key>
                    key for encrypt/decrypt

        -m, --mode <mode>
                    0 for encrypt, 1 for decrypt
```
